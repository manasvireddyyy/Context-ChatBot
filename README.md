# 🤖 Context ChatBot

A Java console chatbot that demonstrates how conversation history can be stored using an `ArrayList`. This project is the first step toward building an AI chatbot that understands context.

## Features

- Stores user messages
- Maintains conversation history
- Uses ArrayList for dynamic storage
- Console-based application

## Technologies

- Java
- ArrayList
- Scanner
- Git & GitHub

## Concepts Learned

- ArrayList
- Enhanced for loop
- Dynamic data storage
- Conversation history
- User input using Scanner

## Current Progress

- ✅ Phase 1 – Learn ArrayList
- ✅ Phase 2 – Store User Messages

Future phases will integrate AI responses and Google Gemini API.
